About
=====

This is a heroku compatible version of the original https://github.com/LeonB/pylibshout
It is meant to be used with https://github.com/ddollar/heroku-buildpack-multi , https://github.com/ddollar/heroku-buildpack-apt and https://github.com/heroku/heroku-buildpack-python


Introduction
------------
Based on the c-libary libshout 2 and built with Cython

Usage
-----
For now, see test/example.py. It's very straightforward.
